1. CMDA — What It Covers and When to Use It (§21.2)

Control Mode Density/Area (CMDA) protects Class I–IV commodities stored in:

  • Palletized, solid-piled, bin box, or back-to-back shelf storage up to 30 ft (non-encapsulated)
  • Shelf storage up to 15 ft (non-encapsulated)
  • Encapsulated commodities up to 15 ft (palletized, solid-piled, bin box, shelf)
  • Encapsulated palletized/solid-piled up to 20 ft — with special criteria
Always start with Chapter 20 → then apply Chapter 21 for CMDA design criteria.

2. New Systems vs. Existing Systems — Critical Distinction (2022 Change)

New systems (§21.2.2.2): Density from Table 21.2.2.2.1 with a fixed design area of 2,000 ft². No curves — just read the table for your commodity class and storage height.

Existing systems — evaluation or modification (§21.2.2.3): Use density curves:

  • Figure 21.2.2.3.1 — Ordinary temperature sprinklers, 20 ft storage
  • Figure 21.2.2.3.2 — High temperature sprinklers, 20 ft storage
  • Figure 21.2.2.3.3 — Height adjustment factor (for heights other than 20 ft)
⚠️ EXAM TRAP: On a new system, area is always 2,000 ft² — never adjusted from the table. The density comes from Table 21.2.2.2.1 based on storage height and temperature rating.

3. The 5-Step CMDA Design Process (Annex A.21.2)

  1. Determine the commodity class
  2. Select density and area from the appropriate figure or table
  3. Adjust the density for storage height (existing systems only — using Figure 21.2.2.3.3)
  4. Increase the operating area by 30% if a dry pipe system is used
  5. Satisfy minimum densities and areas
For dry systems: area × 1.30 — but density stays the same. Never adjust density for dry pipe.

4. Minimum Density and Area Rules (CMDA)

RuleValue
Minimum density floor — storage over 12 ft0.15 gpm/ft²
Minimum remote area — wet pipe2,000 ft²
Minimum remote area — dry pipe2,600 ft²
Dry pipe area increase30% (area only — density unchanged)

Additionally: Class III and IV densities must be at least equal to OH2 (Ordinary Hazard Group 2) for the given area of operation.

5. Special Rules and Exceptions

  • K-11.2+ ordinary/intermediate sprinklers listed for storage → may use high-temperature densities (§21.1.9)
  • Metal bin boxes or closed shelves with face area < 16 ft² → area of application can be reduced by 33%
  • Back-to-back shelf storage 12–15 ft → use Figure 21.2.2.3.1 for design density
  • Encapsulated palletized/solid-piled 15–20 ft → must use K-factor ≥ 11.2 sprinklers; Class I min density = 0.46 gpm/ft² over 2,000 ft²
  • Bin box and shelf storage in Chapter 26 (moved from Ch. 21 in 2022) — walkways at ≤12 ft vertical intervals require sprinklers under each walkway

6. CMSA — Control Mode Specific Application (§22.2)

CMSA sprinklers protect palletized and solid-piled Class I–IV storage only.

NOT permitted for bin box or shelf storage — those require spray sprinklers with CMDA density/area criteria.

System TypeNumber of Design Sprinklers
Wet pipe15 or 20
Dry/Preaction25
Design area = number of sprinklers × max coverage per sprinkler (from Table 13.2.5.2.1). Then: branch line length = 1.2 × √(design area).

7. ESFR — Early Suppression Fast Response (§23.3)

  • Protects Class I–IV and Group A plastics in palletized and solid-piled storage up to 40 ft high
  • NOT permitted for bin box or shelf storage (solid structure blocks discharge from reaching the fire)
  • Wet pipe systems only — no dry, preaction, or deluge
  • Operating pressure: 15 to 75 psi minimum
  • Design area: 12 sprinklers — 4 on each of 3 branch lines
  • K-factors available: 14.0, 16.8, 19.6, 25.2
⚠️ ESFR = wet pipe ONLY. The suppression model depends on instant water delivery — a dry system delay defeats the early-suppression mission entirely.

8. In-Rack Sprinklers in Combination with CMDA (§25.3)

  • Chapter 25 was completely rewritten in 2022 — know the new structure
  • Aisle width + ceiling sprinkler temperature → determine which design curve to use
  • In-rack sprinklers must discharge not less than 15 psi
  • Adding in-rack sprinklers can reduce the required ceiling density
  • More in-rack levels = lower ceiling density requirement
  • Max rack system floor area coverage: 40,000 ft²
  • Separate control valves required for in-rack and ceiling systems (except <20 sprinklers)
Horizontal barriers: sheet metal, wood, or similar — extend full length and depth of rack — used to impede vertical fire spread in combination with in-rack sprinklers.

9. Worked Example — Existing System, Dry Pipe (Annex A.21.2)

Storage: Greeting cards (Class III) | Height: 22 ft | Ordinary temp sprinklers | Dry system

  1. Class III commodity
  2. Figure 21.2.2.3.1 → 0.225 gpm/ft² over 3,000 ft²
  3. Height adjustment (22 ft vs 20 ft base) → Figure 21.2.2.3.3: factor = 1.15 → 0.225 × 1.15 = 0.26 gpm/ft²
  4. Dry system → area × 1.30 → 3,000 × 1.30 = 3,900 ft² (density unchanged)
  5. Minimums met: 0.26 ≥ 0.15 ✓ and 3,900 ≥ 2,600 ✓
Final answer: 0.26 gpm/ft² over 3,900 ft²

10. Worked Example — New System, Dry Pipe (Table 21.2.2.2.1)

Storage: Solid-Piled Class I | Height: 25 ft | High-temp sprinklers | Dry system

  1. Class I commodity
  2. Table 21.2.2.2.1 → 25 ft height, high-temp → 0.20 gpm/ft² at 2,000 ft²
  3. No height adjustment needed for new systems (table is already height-specific)
  4. Dry system → 2,000 × 1.30 = 2,600 ft² (density stays 0.20)
  5. Minimums met: 0.20 ≥ 0.15 ✓ and 2,600 ≥ 2,600 ✓
Final answer: 0.20 gpm/ft² over 2,600 ft²
